 第四版固件更新说明:

【安装arduino后,打开MICROMAKE目录下的MICROMAKE.ino打开】

1.根据之前用户反应的问题，增加了更新详细的注释内容。
2.增加汉化代码关键部分内容，使其更具可读性。
3.更改代码使其适应最新版本的Arduino版本。（下载地址：http://arduino.cc/en/Main/Software）
4.增加了繁体中文版本，适应港澳台用户阅读交流。解决简体版本港台用户显示乱码的问题。(请使用繁体版本)
5.启用版本号内容，为后续持续升级进行区分。
6.默认给Z轴高度改为200，防止误操作撞坏打印头。
7.本固件基于Marlin固件修改，配合销售MICROMAKIE 3D打印机学习套件使用。
8.本固件同时可适用于其他三角洲并联臂类型3D打印机。(通过通读学习以及修改适应)
9.本固件可自由传播复制发布，同时也欢迎更多的朋友一起交流汉化，使其适应新手理解学习。
10.尊重劳动者，使用发布请注明出处。
